LearnHub Portal is a full-stack peer-to-peer learning platform that connects students and instructors in an interactive and scalable environment. The platform enables users to browse courses, manage learning content, and engage through a role-based system.

Built using Angular, ASP.NET Core, and SQL Server, the application focuses on performance, reusable component architecture, and responsive user experience across devices.

🚀 Key Features
🔐 Role-based authentication & Authorization (Students & Instructors)
📚 Course browsing and management system
⚡ Responsive and mobile-friendly UI
🔄 RESTful API integration
🧩 Reusable Angular components and modular architecture
✅ Form validation before backend submission
🚀 Optimized performance and scalable structure

🛠 Tech Stack
Frontend: Angular, TypeScript, HTML5, CSS3, Bootstrap
Backend: ASP.NET Core Web API, Entity Framework Core
Database: SQL Server
Tools: Git/GitHub, Swagger, Postman
